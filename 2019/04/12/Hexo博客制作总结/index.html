Hexo制作博客的技术总结


在b站看的codesheep大佬搭建博客的视频，做一点技术总结。感兴趣的小伙伴可以去b站围观。

传送门：手把手教你从0开始搭建自己的个人博客

博主的电脑是windows系统，Linux和Mac系统的小伙伴大体操作一样，但有些指令不同,推荐看视频。

前期准备


 要下载一些软件，其中包括：git,nodejs,还有hexo框架（这个可以稍后下）

 具体安装方法就不讲了，贴两篇帖子吧：

 安装git的：Windows下Git安装和文件操作


 安装nodejs的：Node.js 安装配置



 完成nodejs的安装后我们开始安装hexo这个博客框架,我们可以通过淘宝的 NPM 镜像来下载，这样速度会比直接下载快一点：

 打开gitBash（基于cmd但更方便一些），输入：


 npm install -g cnpm –registry=http://registry.npm.taobao.org


 等待成功后，可以开始安装hexo框架，继续输入：


 cnpm install -g hexo-cli


 现在我们的前期准备已经完成了，现在可以开始做博客了！

制作博客


 在任意的盘符新建一个文件夹以存放博客的源代码，我这里是f盘里的blog文件夹。在gitBash里键入：  cd f:blog  以进入blog目录下，安装依赖包：
hexo init

 现在可以输入 hexo s  启动本地服务器，在浏览器中输入 http://localhost:4000/  就能够看见hexo默认主题下的博客了，现在里面只有一篇Hello World的文章，那也是默认的。





 我们可以先试着写一篇文章试一试。
